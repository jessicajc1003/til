# Flexbox

How to move a feature (photo, btn etc) to the middle of the screen using `flex`:

```
.profile-avatar {
    width: 100%;
    display: flex;
    flex-direction: column;
    align-items: center;
    position: absolute;
    bottom: 200px;
}
```
