# flex

The flex property allows you to declare flex-grow, flex-shrink, and flex-basis all in one line.
**The `flex` property is different from the `flex` value used for the display property.**

```css
.big {
  flex-grow: 2;
  flex-shrink: 1;
  flex-basis: 150px;
}

.big {
  flex: 2 1 150px;
}
```
